# BestwayForms Wordpress 6.0-6.9+ Plugin

Плагин предоставляется без лицензии. Все права остаются за мной.

## Fullstack Integration: WordPress Plugin + n8n ( Optional ) + AI ( Optional ) + Woocommerce ( Optional )

## Описание проекта

Крутейший плагин для обработки заявок с множества потоков на базе WordPress + Woocommerce через n8n + AI.

Плагин разворачивает себя и WP автоматически. Просто запустить.

Запустил плагин -> сверстал нужные шаблон в папке forms -> Сделал из них шорткоды в плагине -> Пользуешься! 

Если нужно вставить плагин в свой wordpress просто зайди в папку wordpress/pluging и скопируй BestwayForms к себе в проект.

## Технологии

- **WordPress 6.9+** - CMS с кастомным плагином
- **n8n** - платформа автоматизации workflows
- **AI** - Возможность настройки работы с агентами
- **Docker** - контейнеризация
- **MySQL** - база данных WordPress

## Быстрый старт

### 1. Подготовка

Зайти в plugins распаковать Woocommerce, если нужен.

### 1. Запуск

```bash
docker-compose up -d --build
```

### 2. Доступы к сервисам

## WordPress

URL: http://localhost:8000

Админка: http://localhost:8000/wp-admin

Логин: admin / password

## n8n ( Optional )

Если n8n включен. Позаботьтесь о том, чтобы он отдавал json данные (!)

URL: http://localhost:5678 Зайти и импортировать workflows (!!)

Логин: admin / password

n8n Webhook эндпоинты:

POST /webhook/wordpress-lead
- Принимает данные из WordPress при создании новой заявки
- Формат JSON: {name, email, phone, message, wordpress_id, source, timestamp}
